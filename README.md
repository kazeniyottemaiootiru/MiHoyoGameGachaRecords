# 项目介绍
本项目是基于WinUI 3开发的通过游戏缓存下来的链接来访问米哈游服务器中存储的数据（官方只给访问6个月内的数据）从而生成抽卡记录并分析，目前支持简体中文，繁体中文，日语和英语便于分析相关语言下的数据。由于需要对文件进行分析，**目前暂不支持国际服，此问题将在近期修复**，敬请等待。**本程序对游戏内语言有强关联，使用前请先确认您游戏当前语言的情况！**
## 工作原理
程序通过定位获取到缓存下来抽卡链接，依次进行访问直到结束（内容为空），然后根据获取的情况统计5星或S级的垫池数量并存储在本地。**本程序不提供云存储服务，一切数据的存储均在您的设备存储。**
# 构建
本项目使用`.NET 8`开发，构建前请先下载好开发环境，建议使用Visual Studio 2026进行构建并打包，同时自备一个签名的证书文件防止安装失败，由于目标游戏仅在Windows上运行且部分游戏仅支持Windows 10及以上系统运行，所以程序最低运行环境为Windows 10 22H2（~~应该可以往前挪几个版本~~）
## 系统要求
Windows 10 22H2及以上版本
# 安装说明
如果使用Release中的安装包请先安装证书（1年）到本地计算机的受信任的根证书颁发机构中，否则安装失败，接着查看您的操作系统的情况，按照下面的操作方式打开发人员模式以保证安装过程中安装部分依赖项。
<details><summary>Windows 11点击这里</summary>
打开设置，依次点击系统、高级，找到开发人员模式的开关，将其打开即可。
</details>
<details><summary>Windows 10点击这里</summary>
打开设置，依次点击更新和安全，开发者选项，将开发人员模式下的从任意源（包括松散文件）安装应用这个开关打开即可。
</details>

安装完成后如果您的Windows中没有`.NET 8`的运行环境，系统将会弹出提示框，按照提示框的操作下载安装`.NET 8`的运行环境即可。
